# Homebrew Tap for Claude Wall

Mission control for Claude Code — monitor all your AI coding agents from one dashboard.

## Install

```bash
brew tap ogtayhuseynov0/tap
brew install claude-wall
```

## Setup

```bash
claude-wall init    # install Claude Code hooks
claude-wall         # launch dashboard
```

## Update

```bash
brew upgrade claude-wall
```

## Uninstall

```bash
claude-wall uninstall  # remove hooks
brew uninstall claude-wall
brew untap ogtayhuseynov0/tap
```
